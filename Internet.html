How the Internet works?

Even though it seems pure magic, what is really behind is just a formal procedure to establish a straightforward communication between your computer and a web server. As in the ancient times when an embassador went to another kingdom to meet the king, an elaborate protocol where established to keep the relations between both nations in proper manner. Something similar happens when you try to access a web page throughout the internet. A specific protocol is followed by the computers to get and fetch the page that you are requesting. In the next image the basic steps to ask and retrieve a web page are depicted.

The flow goes form left to right and comes back. The actors, by order of appearance are: the User, the Internet and the web Servers.
The series of steps followed to retreive a web page are:

  1. Everything starts when you "feed" an URL in the browser
  2. The URL is resolved by the Internet and a request is sent to the Server which has the page stored
  3. The Server sends back a response with the web page (a HTML document)
  4. The Internet redirects the document directly to your computer
  5. The browser displays the web page on the screen
