Some HTML Tags

Here some examples of HTML tags most used in a web document.....

Comments:
The tag <!--...--> is used to insert comments in the source code, be aware that comments are not displayed in the browsers. You can use comments to explain your code, which can help you when you edit the source code at a later date. This is especially useful if you have a lot of code.

Links:
The <a> tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the <a> element is the href attribute, which indicates the link's destination.

Line breaks:
The <br> tag inserts a single line break. The <br> tag is an empty tag which means that it has no end tag.

Tool Tip:
The <abbr> tag defines an abbreviation or an acronym, like "Mr.","Dec.", "ASAP", "ATM". Marking up abbreviations can give useful information to browsers, translation systems and search-engines. Also for you, if the <abbr> tag is defined with the global title attribute can be used as a "Tool Tip" to show the full version of the abbreviation/acronym when the mouse is positioned over the <abbr> element.

Headers:
The <h1> to <h6> tags are used to define HTML headings. <h1> defines the most important heading. <h6> defines the least important heading.
For example:
<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>
<h4>This is heading 4</h4>
<h5>This is heading 5</h5>
<h6>This is heading 6</h6>

Images:
The <img> tag defines an image in an HTML page. The most important attributes of the <img> tag are src and alt. Note that images are not technically inserted into an HTML page, images are linked to HTML pages. The <img> tag creates a holding space for the referenced image.
