Notes of Lesson #1, The Basics of the Web and HTML

By Alejandro Cadena

Key words of the Lesson:

WWW, Server, HTTP, HTML, URL, Browser, Tag, Element, Attribute


WWW

It has also commonly become known simply as the Web and is an information system of interlinked hypertext documents that are accessed via the Internet. All these documents are made using HTML and those are stored in several locations around the world. The documents are called web pages and are accessed with a computer running a software application called web browser.

Server

A computer Server is a complex thing but for now, we just said that is a computer with a huge capacity where the web pages are collected.

HTTP

Short for HyperText Transfer Protocol. Is the underlying protocol used by the Web and defines how messages, between Servers and browsers, are formatted and transmitted. For example, when a URL is entered in the browser, this actually sends an HTTP command to the Server directing it to fetch and transmit the requested web page.

HTML

Short for HyperText Markup Language. A markup language is a set of markup tags, so HTML is a language for describing web documents (web pages) where those are described by HTML tags. Each HTML tag describes different document content.

URL

Short for Uniform Resource Locator. Is a reference to a resource that specifies the location of the resource on a computer network and a mechanism for retrieving it. URLs occur most commonly to reference web pages.

Browser

The purpose of a browser is to read HTML documents and display them. Be aware that the browser does not display the HTML tags, but uses them to determine how to display the document.

Tag

HTML tags are keywords surrounded by angle brackets. Normally the Tags come in pairs like <p> and </p>, where the first tag in a pair is the start tag (often called the opening tag), the second tag is the end tag (often called the closing tag). The purpose of the end tag is "Close" the content between both tags and tells the browser where the format of the content ends. The end tag is written like the start tag, but with a slash before the tag name.

Element

The HTML element is everything from the start tag to the end tag, namely are written with a start tag, with an end tag, with the content in between: <tagname>content</tagname>

Attribute

Attributes provide additional information about HTML elements and are always specified in the start tag throughtout name/value pairs like: name="value".
